## 📌 About The Project

This portfolio is designed to present my projects, skills, and development work in a clean and interactive way. It focuses on modern UI design, responsiveness, and smooth user experience across all devices.

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Suyash-coder-kumar/portfolio-site.git

# Navigate into the project
cd portfolio-site/backend

# Create virtual environment
python -m venv venv

# Activate it
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run server
python manage.py runserver
```
